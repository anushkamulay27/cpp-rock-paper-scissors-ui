# Rock Paper Scissors – C++ Graphics Game

A graphical Rock–Paper–Scissors game developed in C++ using WinBGIm (graphics.h).
The game provides a mouse-based interactive UI where the user plays against the computer.

## Features
- Graphical user interface using graphics.h
- Mouse click based input
- Randomized computer moves
- Live result display (Win / Lose / Draw)
- Replay and exit support

## Technologies Used
- C++
- WinBGIm Graphics Library
- Mouse & Keyboard Events
- C++ libraries 

## How It Works
1. User selects Rock, Paper or Scissors by clicking buttons.
2. Computer randomly selects its choice.
3. Result is displayed graphically.
4. User can replay or exit.

## Use Case
- Learning graphics programming in C++
- College mini-project
- Game development basic
- First project
